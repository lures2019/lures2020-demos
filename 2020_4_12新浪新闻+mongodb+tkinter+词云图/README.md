### 代码说明：

```
1、采用tkinter图形化界面+MongoDB数据库分别用来展示及存储数据！
2、使用的python的相关库及作用：
	requests:爬虫请求必备
	jieba:中文分词
	time:延时，反爬的小手段
	pymongo:python操作MongoDB数据库
	tkinter：最简单的图形化界面库
	digits:去除字符串中的数字
	WordCloud:python中词云图库
```

### 代码演示要求：

```
1、代码演示前，需要自己创建MongoDB数据库的数据库以及数据表，视频中演示的是：数据库news,数据表news_message
2、目录中的mask.jpg是创建的词云图的形状的模板，可以换成别的库
3、代码演示的是爬取新浪新闻10页数据，存储到MongoDB中，可以换成更多的页数！
```
