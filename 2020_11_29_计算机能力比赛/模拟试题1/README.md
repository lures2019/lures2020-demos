#### 盲点

```
1）'x'模式打开文件读数据，如果文件已存在则会失败
2）Python3完全支持Python2的缩进策略
3）list中的元素是按大小顺序排列的
4）Python2的整型数一般是int，数字超过一定范围后变成long
```
